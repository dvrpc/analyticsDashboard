Reference

A quick reference guide to the analytics provided and what they mean

  • Sessions are groups of user interactions that take place within a given time frame, for example 30 minutes. Sessions contain page views and other analytics. A session begins when a user opens a page and ends when the session time frame passes without any user interaction. Click here for a more in-depth breakdown of sessions.

  • Page Views are recorded every time a page is viewed. Page Views are recorded when a user visits a page, refreshes a page or hits 'back' to return to a page.

  • Unique Page Views are pageviews that are generated by the same user during the same session. Unique page views represent the number of sessions during which that page was viewed one or more times.
